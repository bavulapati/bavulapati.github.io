About me

I'm a software engineer specializing in startups and open source, with expertise in low-level systems, native Node.js addons, and performance-critical application.

  • I'm the original author of the RemotePC Linux App , an Electron.js desktop app with Linux services and native node modules for real-time remote streaming.
  • On the RemotePC R&D team, achieved >30 FPS streaming by offloading the pipeline—desktop capture, RGB-to-YUV conversion, H.264 encoding/decoding—to GPU via hardware acceleration and FFmpeg, slashing resource use.
  • Built and led the RemotePC Linux team from hiring to delivery of a production-grade app.
  • Created IDrive BMR's partner manage remote access using WebRTC and signaling servers.
  • Contributed to Postman's Linux ARM desktop app support. Contributed to postman desktop application's launch performance optimizations by 20%.
  • Maintaining and revamping Ajv-Plus JSON Schema Validator for full JSON Schema compliance.

Available for consulting on C/C++, hardware accelerated video, native Node addons, Ajv-Plus, JSON Schema, architecture. reaching out
